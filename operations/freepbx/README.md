# Projet FreePBX - Système de Téléphonie VoIP

Ce dépôt contient l’ensemble de la documentation opérationnelle de votre serveur **FreePBX**.  
Il a été conçu pour vous permettre de comprendre rapidement le projet, de le reproduire et de l’exploiter au quotidien.

### Contenu du dépôt

- **README.md** → Ce fichier (vue d’ensemble du projet)
- **INSTALLATION.md** → Guide complet de l’installation initiale
- **CONFIGURATION.md** → Guide détaillé de la configuration (firewall, extensions, softphones)
- **backup.md** → Procédures de sauvegarde et restauration
- **monitoring.md** → Surveillance et monitoring du serveur
- **troubleshooting.md** → Résolution des problèmes courants
- **images/** → Toutes les captures d’écran utilisées dans les documentations

### Description du projet

Vous avez installé et configuré un serveur FreePBX 16 permettant :
- La gestion de postes internes (extensions 1000, 1001, 1003, etc.)
- L’enregistrement de softphones 3CXPhone
- Un firewall sécurisé (Sangoma Smart Firewall + Responsive Firewall)
- Des appels internes fonctionnels

Le serveur est accessible à l’adresse : **http://10.60.70.5/admin**

---

**Pour commencer :**  
1. Consultez **INSTALLATION.md** si vous souhaitez reproduire l’installation depuis zéro.  
2. Consultez **CONFIGURATION.md** pour reproduire la configuration actuelle.  
3. Consultez **backup.md**, **monitoring.md** et **troubleshooting.md** pour les opérations courantes.
